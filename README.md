# Gaming Platform Player Transaction Analysis 🎮📊

## 📌 Project Overview
This project analyzes player transactions from a gaming platform using Excel.  
The goal is to uncover spending patterns, player behavior, and business insights.

## 🗂 Dataset
- Features include:
  - Player ID
  - Transaction Amount
  - Transaction Date
  - Game Type
  - Payment Method

## 🔍 Analysis Performed
- Data Cleaning & Formatting
- Descriptive Statistics
- Pivot Table Analysis
- Trend Analysis by Date
- Player Segmentation
- Top Spenders Identification
- Visualization using Excel Charts

## 📈 Key Insights
- Average transaction value trends
- Most popular payment methods
- Revenue contribution by top players
- Seasonal/monthly spending patterns
- Suspicious Activity 

## 🛠 Tools Used
- **Microsoft Excel** (Pivot Tables, Charts, Conditional Formatting)

## 🚀 How to Use
1. Download the Excel file.
2. Open in MS Excel or Google Sheets.
3. Explore pivot tables & charts.
